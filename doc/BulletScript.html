BulletScript language reference

BulletScript uses two fundamental objects: controller and emitter. Both are simple state machines, but they differ slightly in how they can be controlled, and what they control themselves. Emitters are responsible for actually creating objects, and for controlling the behaviour of those objects afterwards. Controllers are used to group and control emitters, synchronising them.

Emitters

Emitters are defined in the following form:

emitter emitter_name
{
	[member_variable_list]
	[affector_instance_list]
	[function_list]
	state_list
}
